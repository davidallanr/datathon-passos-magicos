# Datathon Passos Mágicos

Projeto desenvolvido no Tech Challenge da FIAP.

Objetivo: analisar indicadores educacionais e desenvolver um modelo preditivo capaz de identificar alunos em risco educacional.

## Tecnologias utilizadas

Python  
Pandas  
Scikit-learn  
Streamlit  

## Aplicação

A aplicação permite inserir indicadores educacionais e prever o risco educacional do aluno.
